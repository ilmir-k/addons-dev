Print certain products on Order Printers

Specify certain products to be allowed to print on order printers

By default, you can assign product categories only to be printed on order printers and it is not convenient in some cases, e.g. if you want to print some products on one printer and other products of the same category on another one. The module eliminates this disadvantage and allows to specify certain products, which should be printed on order printers. Note that in this case it is not required to specify product categories.

How It Works

  • Go to Point of Sale → Configuration → Order Printers menu and open the order printer form
  • You can see new field Printed Products
    •

Click on "Add an item" and select products, which should be printed on order printer

Finally, open POS session, add products allowed to print and click on "Order" button
