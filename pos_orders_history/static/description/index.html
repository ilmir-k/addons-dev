POS Orders History

See all paid orders from special menu in POS

How It Works

Open POS session. Here you can see new History button.


Once the button is clicked, all paid orders are shown. You can use the filters to be able to easily find the orders needed.

To see the details, click on order item.


Go to Point of Sale → Configuration → Point of Sale and open POS configuration form. See new options:

  • Orders History - activate/deactivate History button
  • Current Day Orders Only - show orders for current day only
  • Show Cancelled Orders - show cancelled orders in addition to paid orders
